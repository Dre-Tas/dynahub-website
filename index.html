Easy deployment for Dynamo content

Track changes. Secure your IP. Collaborate better. Make Dynamo more accessible.

Get Started

Do you already have a GitHub account?

If you don't, this is where you should start.

DynaHub is extension for Dynamo that allows you to seamlessly connect to GitHub. Upload your graphs online to be able to reach them from anywhere.

Host your Dynamo content on the most used Git provider

Once you created and account, create a new repo, public or private and start populating it with your Dynamo files. Every time you edit them you'll see what changed and you'll be able to roll back to a previous (hopefully working) version.

Deploy your graphs everywhere in the world!

Access your GitHub repository without leaving Dynamo and open your files with just a click! No worries about messing up and unwanted changes.
